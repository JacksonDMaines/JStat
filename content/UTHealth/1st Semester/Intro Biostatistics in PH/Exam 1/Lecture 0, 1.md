## Types of Variables
**Numerical (Quantitative):** 
- Continuous
	- Between two values there exists a meaningful number
	- Think age, weight, ....
- Discrete
	- No meaningful number between to values
	- Think rolling dice, number of people in room, ....
 
**Categorical (Qualitative):** 
 - Ordinal
	 - Has a meaningful order
	 - Think economic class, satisfaction rating, ...
 - Nominal
	 - Has no meaningful order
	 - Think eye color, race, .....
