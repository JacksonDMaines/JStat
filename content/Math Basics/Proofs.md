
## Induction

Let  $p(n)$, $\forall n\geq n_{0}$ and $n,n_{0}\in\mathbb{z}_{+}$ We need to show that $p(n)$ is true for all possible values of n.
- 1st: Show $p(n)$ is true for the smallest possible value of n. $p(n_{0})$ 
- 2nd: Assume the statement is true for $n = k$, for some integer $k\geq n_{0}$. Show the statement is true for $n=k+1$
